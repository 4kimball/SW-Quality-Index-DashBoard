# S/W 품질지표 Dash Board 시스템 개발

> - 다양한 S/W 시스템 및 모듈에서 전송된 품질 Raw데이터를 활용하여 전사 S/W 품질지표 현황을 한 눈에 확인 가능한 Dash Board 개발



### :bulb: 주요 기능

- 전체 / 시스템 / 팀별 지표 조회
- 통계 그래프 사용
- 기준 미달 지표에 대한 하이라이트 표시
- S/W관련 용어 설명
